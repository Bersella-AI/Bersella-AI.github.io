公司背景

羊驼，原产于高山峭壁、热带草原 南美洲；其因憨厚可掬的外表、可爱的叫声（尤其在幼年期）、温顺的性格，以及卓越的载重能力，受到喵窝内外人们的喜爱。自去年七月，无限首次驯养羊驼以来，无限家中的羊驼队伍日益壮大，并开始对外输出羊驼；在无限家兼作“羊驼乐园总部”的同时，于琥珀川北沼区设置分园，联络两地居民之间的感情。

但经年累月的粗放扩张，使得家中院子混乱不堪；科学管理之缺失，使得劣质羊驼泛滥。与此同时，在家以北的烂尾楼内露天暂居的家兔，亦面临雷击威胁，亟需一处安全、舒适的新居。遂决定就近修建动物园 养殖大棚 繁育基地容纳之；并在此基础上成立公司，科学、规模化管理动物。同时，公司期待能够利用独特优质资源，培育出优良品种的羊驼。

参观及业务往来须知

羊驼、家兔养殖

（以下将羊驼强度，即羊驼载货空间的三分之一，称为“星”。）

  • 我司培育并供应四种花色、3星及以上之羊驼；
  • 我司培育并供应六种花色的家兔（不计杀手兔、命名“Toast”者）。
  • 我司通过“羊驼发送站”供应羊驼，向外界开放领养。对于个人或集体，领养规则如下：
    • 目前同时开放3星、4星羊驼的领养。其中：
      • 4星当前尚未出栏，即日起逐步开放；
      • 3星将逐步淘汰。
    • 暂定领养任意只均不收费；
    • 领养数量不大于4只——可随时运出，无需向我司报告（当然，建议报告）；
    • 领养数量超过4只——可随时运出，但领养前需向我司报告领养数量；
      • 特别地，领养总数超过12只，或领养任意一种花色超过5只——请联系我司面谈。
    • 取名为Сагири Изуми（和泉纱雾）的羊驼不可带走，也不可重命名。
      • （该俄文名称实际应为Сагири Идзуми。）
    • 请勿在我司设施内伤害羊驼。
      • 如不慎造成伤害，请迅速以拴绳制服之。
      • 如造成伤害，导致其攻击另一羊驼，请马上以拴绳或“鬼手”将双方拉开距离。
      • 如造成其死亡，请将其计入羊驼领养数。
  • 家兔供应需联系我司预约。
  • 未经我司许可，禁止私自从繁育基地运出牲畜。

生鲜物流

  • 在公共铁路网（包括下界）可及之处（车站向外延伸256米），无偿提供羊驼押运服务；
    • 若目的地距离最近的车站超过256米，视当地地形和客户素质考虑是否延长护送距离。
  • 暂不受理家兔押运申请，将来考虑有偿押运。

旅游业

  • 在羊驼乐园，和羊驼零距离逗趣。
  • 繁育基地一层（羊驼）仅限在外围参观。
  • 繁育基地二层（家兔）开放参观。
    • 由于家兔生性怕人，为避免其出逃，参观时请严格遵从当地告示牌之指示。
    • 临时接纳北极熊。仅供观赏，严禁触碰。
  • 总部官邸内也允许参观。
    • 请勿触碰猫、狗、鹦鹉等宠物。
  • 考虑于繁育基地旁设置北极熊园。
  • 考虑于总部附近再新建羊驼乐园。

农牧及加工产品零售

我司计划提供以下产品：

  • 农业产品（花卉）
  • 畜牧产品（皮革、牛肉、兔肉、兔子脚等）
  • 简单加工品（书架、牛排、拴绳等）

可在总部官邸内的一处角落，以及上述“自销店”内立即进行交易。
